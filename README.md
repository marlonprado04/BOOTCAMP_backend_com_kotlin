# Bootcamp: Backend com Kotlin

Repositório para armazenar anotações e links referentes ao Bootcamp de backend com Kotlin oferecido pela Digital Innovation One.

## O que vão encontar no repoisitório

Tentarei seguir e manter a estrutura do curso na estrutura do repositório, ou seja, criar uma pasta para armazenar os conteúdos referentes a cada etapa do bootcamp.

Como o bootcamp é composto por **cursos**, **desafios** e **lives** que podem fazer parte de algum outro bootcamp já realizado anteriormente podem existir pastas simbólicas que serve para armazenar o link de onde o conteúdo original se encontra.

Por exemplo, a pasta número "02" que se refere ao curso de Git e GitHub foi realizado em outro bootcamp, então dentro haverá apenas um README simbólico com o link para o repositório onde as anotações se encontram.

Dessa forma, consigo manter a estrutura do bootcamp para poder mensurar o progresso de aprendizado também criar referências úteis.

Dito isso, abaixo você vai encontrar a lista resumida de conteúdos do Bootcamp para servir como um norte deste bootcamp.
